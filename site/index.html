Как подключиться

  1. Установите Streisand или V2Box из App Store
  2. Откройте приложение → «+» → «Импорт из буфера обмена» (или отсканируйте QR-код камерой)
  3. Вставьте ссылку подписки (обновляется автоматически) или VLESS-ключ
  4. Нажмите на профиль → «Подключиться». При первом запуске подтвердите добавление VPN-конфигурации в настройках iOS

  2. Откройте приложение → «+» → «Добавить подписку» (или отсканируйте QR)
  3. Вставьте ссылку подписки или VLESS-ключ
  4. Выберите профиль и нажмите кнопку подключения. Разрешите создание VPN-подключения

  2. Установите и запустите Hiddify
  3. Скопируйте ссылку подписки → в Hiddify нажмите «Добавить из буфера»
  4. Переключите тумблер в положение «Подключено»
  1. Установите V2Box из Mac App Store или Hiddify с GitHub
  2. Откройте приложение → «+» → «Импорт из буфера обмена»
  3. Вставьте ссылку подписки или VLESS-ключ
  4. Нажмите «Подключиться» и разрешите добавление VPN-профиля в macOS
Совет: используйте ссылку подписки, а не сырой VLESS-ключ — тогда при смене сервера или обновлении параметров клиент подтянет изменения сам.
